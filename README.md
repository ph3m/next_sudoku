# 🧩 Sudoku Game

A simple, interactive Sudoku game built using **HTML, CSS, and JavaScript**.
This project reflects my passion for **gaming logic, problem-solving, and frontend development**.

---

## 🎮 About the Project

This is a browser-based Sudoku game where players can solve randomly generated puzzles directly in the browser.

It focuses on:

* Logic-based gameplay
* Clean UI design
* Interactive user experience using vanilla JavaScript

## 🛠️ Tech Stack

* HTML5 – Structure
* CSS3 – Styling & layout
* JavaScript (Vanilla JS) – Game logic & interactivity

## ✨ Features

* 🎲 Sudoku puzzle grid (9×9)
* 🧠 Interactive cell input
* ✔️ Basic validation system
* 🔄 Restart/New game option
* 📱 Responsive design (mobile-friendly)
* 🎮 Simple and clean UI for better gameplay
